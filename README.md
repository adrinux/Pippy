# Pippy

A Hugo theme based on H5bp and including hugo pipes support

## Dependencies

### PostCSS

Install with npm:

- postcss-cli
- postcss-import
- postcss-preset-env
- autoprefixer
- postcss-reporter

``` bash
npm install -g postcss-cli
```


``` bash
npm install -D postcss-preset-env autoprefixer postcss-reporter postcss-import
```
